Decentralized Supply Chain Simulation

This project implements a blockchain-based supply chain in C++, using hashing to securely verify transactions. The system tracks product movement across nodes, logs events in an immutable blockchain, and includes attack simulations to test security. The project explores decentralized validation methods to enhance supply chain traceability and security.

Sorting Algorithms

This project compares the performance of multiple sorting algorithms, including Quick, Merge, Heap, and Bubble Sort, across different dataset types (sorted, reverse-sorted, random, and duplicates). Implemented in C++, it measures execution time and efficiency, visualizing results using Python and R.
